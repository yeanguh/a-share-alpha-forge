# A股通用估值分析框架

## 绝对估值法

### 1. DCF 现金流折现法（Discount Cash Flow）

**核心逻辑**：企业价值等于未来所有自由现金流的折现之和。

**计算公式**：
```
企业价值 = Σ(FCF_t / (1 + r)^t) + 终值 / (1 + r)^n
```
- FCF = 自由现金流（经营现金流 - 资本支出）
- r = 折现率（通常用WACC加权平均资本成本）
- 终值 = 稳定增长阶段的企业价值

**适用场景**：
- 现金流稳定可预测的成熟企业
- 公用事业、消费必需品等行业
- 需要长期价值判断时

**局限性**：
- 对增长率和折现率假设非常敏感
- 成长型企业未来现金流预测误差大
- 计算复杂，需要较多假设

### 2. EBITDA倍数法

**核心逻辑**：EV/EBITDA 估值，剔除资本结构、折旧摊销影响，更反映企业经营本身价值。

**计算公式**:
```
EV = 市值 + 净负债
EV/EBITDA = 企业价值 / 年度EBITDA
```
- EBITDA = 息税折旧摊销前利润

**适用场景**:
- 重资产行业（制造业、电信、能源）
- 并购估值常用
- 对比不同资本结构企业

**优势**:
- 不受折旧政策影响
- 不受所得税差异影响
- 适合对比同行业不同公司

## 相对估值法体系

### 1. PE（市盈率）- 核心指标

**计算公式**：
```
PE = 当前股价 / 每股收益(EPS)
```

**适用场景**：
- 盈利稳定的成熟企业
- 消费、医药、科技等行业

**估值判断**:
- PE低于行业下限 → 低估，具备安全边际
- PE在行业合理区间内 → 合理估值
- PE高于行业上限 → 高估，需超预期增长支撑

### 2. PB（市净率）- 安全边际指标

**计算公式**:
```
PB = 当前股价 / 每股净资产(BVPS)
```

**适用场景**:
- 银行、保险、地产等重资产/金融行业
- 周期行业判断底部
- 亏损企业辅助判断

**估值判断**:
- PB < 1 → 破净，通常市场对前景极度悲观
- 低PB + 高ROE → 潜在价值低估机会

### 3. PS（市销率）- 成长辅助指标

**计算公式**:
```
PS = 当前市值 / 营业收入 = 当前股价 / 每股营收
```

**适用场景**:
- 高成长尚未盈利的企业
- 营收增长快但利润被投资/研发费用压制的企业

**特点**:
- 不会出现负值，对亏损企业依然有效
- 受毛利率影响大，不同行业差异显著

## 业绩预期法

### 核心逻辑

合理估值 = 未来12个月一致预期EPS × 行业合理PE中枢

### 使用步骤

1. 获取券商一致EPS预期（当年或下一年）
2. 确定对应行业合理PE区间
3. 计算合理估值区间和中枢
4. 和当前股价对比，判断上涨/下跌空间

## 券商一致性预期

- 多家头部券商研报的目标价汇总
- 取中位数比平均数更能抵御极端值影响
- 一致性预期反映市场一致观点，但不代表一定正确，需独立判断

## 完整分析流程

```
1. 获取实时股价 → mootdx/通达讯，失败时降级到腾讯财经公开行情和东方财富公开行情
2. 获取估值和基本面数据 → 腾讯/东方财富 PE、PB、市值；AKShare/CNINFO 补财务和公司资料
3. 多维度估值计算：
   - PE/PB/PS 相对估值
   - EV/EBITDA 倍数估值（如有数据）
   - DCF 绝对估值（如需深度分析）
4. 对比行业合理区间，给出判断
5. 结合一致预期计算合理估值中枢
6. 给出分析结论和空间测算
```

## 注意事项

1. **动态调整**: 行业合理PE区间随市场情绪、利率环境变化
2. **个股差异**: 同行业不同个股因龙头地位/成长性不同，PE溢价不同
3. **周期行业**: 周期高点PE低，周期低点PE高，需结合PB判断
4. **成长行业**: 高PE反映高增长预期，需关注增速能否持续匹配
